# Amazon Fine Food Reviews Sentiment Analysis

A Machine Learning and NLP project that performs sentiment analysis on Amazon Fine Food Reviews using supervised learning techniques. The model predicts whether a customer review is positive or negative based on the review text.

---

## 📌 Project Overview

This project focuses on analyzing customer sentiments from Amazon Fine Food Reviews using Natural Language Processing (NLP) and Machine Learning techniques.

The workflow includes:
- Data cleaning and preprocessing
- Text vectorization using TF-IDF
- Model training using classification algorithms
- Performance evaluation using accuracy and other metrics

The final model achieves over **85% accuracy** on sentiment prediction.
